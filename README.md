# Nome do projeto

> Nosso projeto visa entregar ao usuário uma experiência única e eficiênte sobre o uso de uma calculadora para suprir suas necessidades diárias, em diferentes cenários que for desejado.

### Ajustes e melhorias

O projeto ainda está em desenvolvimento e as próximas atualizações serão voltadas nas seguintes tarefas:

- [] Implementar um sistema que armazena os dados calculados para que seja possível consulta-los pelo usuário.
- [] Implementar mais funções, como as de uma calculadora ciêntifica. Tornando assim seu uso mais completo.

## 💻 Pré-requisitos

Antes de começar, verifique se você atendeu aos seguintes requisitos:

- Você deve possuir um sistema operacional Windows, Linux ou Mac.
- Você deve possuir o JDK e a JVM instalados em seu sistema.
- Você deve possuir conhecimentos básicos para a configuração e compição da aplicação em seus sistema.

## 🚀 Instalando <RPG Textual>

Para instalar o projeto, siga estas etapas:

1. Clone o repositório em sua máquina:
2. Abra o terminal e navegue até a pasta do projeto.
3. Execute o comando `javac Main.java` para compilar o projeto.
4. Execute o comando `java Main` para rodar o projeto.`

## ☕ Usando <Calculadora>

Para usar o projeto, siga estas etapas:

1. A interface principal da calculadora é composta por um menu de oções, onde o usuário escolhe uma operação matemática de sua escolha, dentre as opções disponíveis.
2. O usuário pode executar diferentes operações em diferentes cenáriso distintos, sendo eles a adição, subtração, multiplicação e divisão.
3. Durante o uso da calculadora, os dado não ficam armazenados em nenhum lugar, por tanto recomenda-se a anotação das informações obtidas.

## 📫 Contribuindo para RPG Textual

Para contribuir com projeto, siga estas etapas:

1. Bifurque este repositório.
2. Crie um branch: `git checkout -b <nome_branch>`.
3. Faça suas alterações e confirme-as: `git commit -m '<mensagem_commit>'`
4. Envie para o branch original: `git push origin <nome_do_projeto> / <local>`
5. Crie a solicitação de PR.

Como alternativa, consulte a documentação do GitHub em [como criar uma solicitação pull](https://help.github.com/en/github/collaborating-with-issues-and-pull-requests/creating-a-pull-request).
